# MBA NZANG S.L.

MBA NZANG S.L. es una empresa dedicada a democratizar el acceso a la tecnología en Guinea Ecuatorial, formada por un equipo joven y ambicioso de profesionales apasionados por la innovación digital. Ofrecemos soluciones adaptadas a las necesidades de cada cliente, combinando talento local e internacional.

## Tecnologías utilizadas

- **HTML5**: Estructura y contenido de las páginas web.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Interactividad y funcionalidades dinámicas.
- **Font Awesome**: Iconografía utilizada en la web.
- **Netlify**: Despliegue estático (según configuración incluida).

## Estructura del Proyecto

- **index.html**: Página principal de la web.
- **servicios.html**: Detalle de los servicios ofrecidos (diseño web, tiendas online, blogs, etc.).
- **empresa.html**: Información sobre la empresa y el equipo.
- **proyectos-galeria.html**: Galería de proyectos realizados.
- **proyecto-detalle.html**: Detalle de proyectos destacados.
- **productos.html**: Catálogo de productos.
- **contacto.html**: Formulario y datos de contacto.
- **gracias.html**: Página de agradecimiento tras el contacto.
- **hotel-detalle.html**: Detalle de hoteles (si aplica).

### Carpetas
- **img/**: Imágenes y logotipos.
- **js/**: Scripts JavaScript (`script.js`, `proyectos.js`, `hoteles.js`).
- **styles/**: Hojas de estilo CSS (`main.css`, `proyectos.css`, `hoteles.css`).
- **video/**: Recursos de video.

## Instalación y Uso

No se requiere instalación especial. Simplemente abre `index.html` en tu navegador para visualizar la web. Para desarrollo, puedes editar los archivos HTML, CSS y JS según tus necesidades.

## Despliegue

El proyecto puede desplegarse en cualquier servidor web estático. Incluye archivos de configuración para Netlify (`netlify.toml`) y despliegue (`windsurf_deployment.yaml`).
