Descripción del Proyecto

Guinea Ventas es una plataforma de compra-venta y anuncios clasificados para Guinea Ecuatorial. Permite a los usuarios publicar anuncios, chatear, recibir notificaciones y gestionar su perfil de manera sencilla y segura. La aplicación está disponible para Android y próximamente para iOS.

  • Publicación de anuncios clasificados
  • Chat integrado entre usuarios
  • Panel de notificaciones
  • Gestión de perfil y datos personales
  • Diseño moderno y fácil de usar
  • Disponible en versión web y app Android
